
# iOS Guided Access Setup Guide

Guided Access is an iOS feature that locks your device to a single app, preventing children from exiting or accessing other features. This is the **recommended way** to use Baby Play Pad for maximum security.

## What is Guided Access?

Guided Access:
- ‚úÖ Locks the device to Baby Play Pad
- ‚úÖ Disables home button/gesture
- ‚úÖ Prevents access to other apps
- ‚úÖ Disables notifications
- ‚úÖ Can disable touch in specific areas
- ‚úÖ Can set time limits

## Setup Instructions

### Step 1: Enable Guided Access

1. Open **Settings** on your iPhone or iPad
2. Tap **Accessibility**
3. Scroll down and tap **Guided Access**
4. Toggle **Guided Access** to ON (green)

### Step 2: Set a Passcode

1. In Guided Access settings, tap **Passcode Settings**
2. Tap **Set Guided Access Passcode**
3. Enter a 6-digit passcode
4. Re-enter to confirm

> ‚ö†Ô∏è **Important**: This passcode is different from the app's PIN. Don't forget it!

### Step 3: Enable Accessibility Shortcut (Optional)

1. In Guided Access settings, toggle **Accessibility Shortcut** to ON
2. This allows you to triple-click the side button to start/stop Guided Access

### Step 4: Configure Options (Optional)

In Guided Access settings, you can configure:
- **Display Auto-Lock**: Prevent screen from sleeping
- **Touch**: Allow or disable touch
- **Motion**: Allow or disable device motion
- **Keyboards**: Allow or disable keyboards
- **Time Limits**: Set session time limits

## Using Guided Access with Baby Play Pad

### Starting a Guided Access Session

1. **Open Baby Play Pad**
2. **Triple-click** the side button (iPhone X and later) or home button (older iPhones)
   - On iPad: Triple-click the top button
3. The Guided Access screen appears
4. (Optional) Circle areas to disable touch
5. Tap **Start** in the top-right corner
6. Guided Access is now active!

### During Guided Access

- The child cannot exit Baby Play Pad
- Home button/gesture is disabled
- Notifications are hidden
- Control Center is disabled
- The device is locked to the app

### Ending a Guided Access Session

1. **Triple-click** the side button (or home button)
2. Enter your **Guided Access passcode** (not the app PIN)
3. Tap **End** in the top-left corner
4. You can now exit the app normally

## Tips and Best Practices

### For Best Results

‚úÖ **Always use Guided Access** when giving the device to a child
‚úÖ **Set a different passcode** for Guided Access than the app PIN
‚úÖ **Test it first** before giving to your child
‚úÖ **Enable Display Auto-Lock** to prevent screen timeout
‚úÖ **Charge the device** before long play sessions

### Common Scenarios

**Scenario 1: Quick Play Session**
- Start Guided Access
- Let child play
- End Guided Access when done

**Scenario 2: Extended Play**
- Enable Display Auto-Lock in Guided Access settings
- Set a time limit if desired
- Start Guided Access
- Device will alert when time is up

**Scenario 3: Multiple Children**
- Use Guided Access for each session
- Clear canvas between children (shake device)
- Restart Guided Access for next child

## Troubleshooting

### Can't Start Guided Access
- Make sure it's enabled in Settings ‚Üí Accessibility
- Try restarting your device
- Check that you're triple-clicking correctly

### Forgot Guided Access Passcode
- You'll need to reset it:
  1. Go to Settings ‚Üí Accessibility ‚Üí Guided Access
  2. Tap Passcode Settings
  3. Tap Change Guided Access Passcode
  4. Enter device passcode to reset

### Child Exited Guided Access
- Make sure you're using the correct triple-click gesture
- Check that Guided Access is properly enabled
- Verify the passcode is set

### Screen Keeps Turning Off
- Enable Display Auto-Lock in Guided Access settings
- Or adjust Auto-Lock in Settings ‚Üí Display & Brightness

## Alternative: Without Guided Access

If you choose not to use Guided Access, Baby Play Pad still has built-in protection:
- Triple-tap top-right corner to exit
- Enter 4-digit PIN to confirm
- This provides basic protection but is less secure than Guided Access
